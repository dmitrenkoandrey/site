Содержание документа Git

Для создания документа HTML5 нам нужны в первую очередь два элемента: DOCTYPE и html.
Элемент doctype или Document Type Declaration сообщает веб-браузеру тип документа.
!DOCTYPE html указывает, что данный документ является документом html и что используется html5,
а не html4 или какая-то другая версия языка разметки.
